### 프로젝트 소개
Title: MSFC (**Measure Speed For Club**)

```
저희 프로젝트는 인공지능을 이용하여 골프채의 속도를 측정하는 서비스입니다.
yolo v3를 사용하여 골프채를 인식하고 스윙 동작에서 클럽헤드의 좌표를 통한
계산으로 속도를 측정하였습니다.
```
